== Description ==
SMS Alert for Contact Form 7 is a plugin for the popular contact form 7. So this is an extension for the CF7 plugin to send SMS alert. This plugin can send SMS to both admin and user.
This plugin uses SMS API. There are four SMS API uses in this plugin. These are [Twilio](https://www.twilio.com/), [Nexmo](https://www.nexmo.com/), [MSG91](https://msg91.com/), [Textlocal](https://www.textlocal.in/).
In case of MSG91 SMS API, you must have to buy the transactional SMS plan.


== Free Version Features ==
The free version of this plugin comes with the features that are given below:

Can send SMS to both admin and user.

In this plugin, you can enable or disable SMS for admin or user or for both.

This plugin uses 4 SMS API providers. These are Twilio, Nexmo, MSG91 and Textlocal. Choose the one that you like to use.

You can customize the SMS text.

You can put any field from Contact Form 7 just by the field name and it will appear in the SMS text. For example, if you use [field=your-name] in SMS text, it will show the name of the user if the field name "your-name" is available in the contact form.
